    Vorverarbeitung von Text für Machine Learning in node.js

    Für die Klassifzierung von Texten auf Wortebene, ist die Vorverarbeitung eine wichtige Grundlage. Hierfür haben wir einen OpenNLP Wrapper für node.js verwendet. Ein solider Wrapper, jedoch tokenisiert dieser nur sequentiell, sodass wir einen neuen OpenNLP Wrapper geschrieben haben, der parallel arbeitet.
